# LOL zh_CN Launcher

## 在右边 Release 里面下载，双击打开即可使用。

英雄联盟简体中文启动器，适配于 4/29 更新过后的 Vanguard 反作弊引起的不能使用快捷方式更改的问题

关于安全性 ：完全绿色安全，LOL 启动后自动退出。程序使用 pyinstaller 打包无加密无混淆，对安全有顾虑可以自行解包查看，或者自己使用 python 解释器运行。
